About this project

This academic project was created with the aim of reinforcing the knowledge acquired in the use of the technologies learned. The project is based on the creation of a social network where users can find and share information about collection points for bulky waste in their localities.

Tecnologies: HTML5, CSS3, JavaScript

Creation period: 2025
